# MedicalRecordSharing
Medical Record sharing using existing Block Chain Network

## Team Member
* Vinay R Hasan (20162041)

### Install application
```bash
git clone --recursive https://github.com/suryanshagnihotri/MedicalRecord.git
cd MedicalRecord/server
npm install
```
### Start application
```bash
cd MedicalRecord
node server/app.js
```
